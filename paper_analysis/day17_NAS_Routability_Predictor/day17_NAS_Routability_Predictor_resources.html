5. 复现指南

5.1 核心模块复现步骤

# 1. 克隆 ProxylessNAS 作为基础框架
git clone https://github.com/MIT-HAN-LAB/ProxylessNAS.git
cd ProxylessNAS

# 2. 安装依赖
pip install torch torchvision
pip install -r requirements.txt

# 3. 自定义搜索空间（需自行实现）
# - 修改 proxylessnas/ops.py 添加：
#   · 空洞卷积 (atrous conv, dilation=2)
#   · 混合卷积 (MixConv, kernel [7,9,11,13])
# - 修改 model_builder.py 定义 Guide-DAG 结构
# - 实现子图采样算法 (Algorithm 2)
# - 实现边权更新算法 (Algorithm 1)

# 4. 特征提取（需自行实现）
# - 17 通道特征: 5 单元密度 + 12 线密度
# - 需 Cadence Innovus 或 OpenROAD 生成布局数据

# 5. 搜索
python search.py --task violated_net_count  # 或 drc_hotspot

# 6. 训练最终模型
python train.py --arch <nas_output_arch>

5.2 关键实现注意事项

模块实现要点
Guide-DAG7 顶点全连接 DAG，每个顶点对应一个候选操作
伪顶点添加 v_0 代表下采样层，连接到所有其他顶点
子图采样Softmax 归一化边权 → 概率采样 → 构建 sampled-DAG
边权更新w_e = w_e · exp(α(μ - β))，β = top-5 平均
特征提取224×224 分辨率，17 通道，含 4 种飞行线
训练超参Adam, lr=0.0005, batch=48, 45 epochs, L2=1e-5
