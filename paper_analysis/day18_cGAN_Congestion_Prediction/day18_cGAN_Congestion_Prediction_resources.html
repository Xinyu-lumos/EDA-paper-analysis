Key Architecture Summary

Input: x = stack(im_place, im_connect)  in  R^(256x256x4)

Generator (FCN with skip connections):
  Encoder: Conv1..Conv8  (256x256x4  -->  1x1x512)
  Decoder: Deconv1..Deconv8 (1x1x512  -->  256x256x3)

Discriminator:
  Conv1..Conv5 + Sigmoid  (256x256x6  -->  30x30x1)

Loss: cGAN_loss + lambda * L1_distance  (lambda = 50, beta = 0.1)
Optimizer: Adam (lr=0.0002, beta1=0.5, beta2=0.999, eps=1e-8)
Training: 250 epochs, batch size 1, 2-3 hours on 1080Ti
Inference: ~0.09 seconds per image
