组件详情

边类型topo_in, topo_out (网表), grid_edge (布局), geom_loc, geom_bb (空间映射)

Grid 特征type_embed(1) + h_density(1) + v_density(1) + rudy(1) + pin_density(1)
Grid 标签chanx_l4, chany_l4, chanx_l16, chany_l16 (4维)

参数默认值
层数2 层 HeteroGraphConv
隐藏维度64
网表消息传递GraphSAGE (pool aggregator)
布局消息传递GraphSAGE (pool aggregator)
空间映射消息传递GCN
聚合方式mean (sum 和 stack 可选)
注意力头数 (GAT)4
激活函数LeakyReLU (投影), ReLU (层间)

4. 关键代码片段

4.1 FHGNN 前向传播（source/model/FHGNN.py）

def forward(self, graph):
    # 两阶段特征投影
    b_h = F.leaky_relu(self.lin_block(graph.nodes['block'].data['feature']))
    n_h = F.leaky_relu(self.lin_net(graph.nodes['net'].data['feature']))
    g_h = F.leaky_relu(self.lin_grid(graph.nodes['grid'].data['feature']))

    graph.nodes['block'].data['h'] = F.leaky_relu(self.lin_block_2(b_h))
    graph.nodes['net'].data['h'] = F.leaky_relu(self.lin_net_2(n_h))
    graph.nodes['grid'].data['h'] = F.leaky_relu(self.lin_grid_2(g_h))

    # 多层异构图卷积
    for i in range(self.layer_num):
        h = self.gnn_layers[i](graph, graph.ndata['h'])
        h = {k: F.relu(v) for k, v in h.items()}

    grid_embed = h['grid']
    pred_out = self.out_lin(h['grid'])
    return pred_out, grid_embed

4.2 g_emb2image 格式转换（source/utils.py）

def g_emb2image(grid_embedded):
    # grid_embedded: [16384, 64] → [1, 64, 128, 128]
    image = grid_embedded.view(-1, 128, grid_embedded.shape[1])\
                        .permute(2, 0, 1).unsqueeze(0)
    return image
